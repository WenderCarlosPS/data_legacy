Prouni
2. Modelo de "Afinidade de Curso" (Recomendação/Classificação)
Objetivo: Prever qual a área de estudo (Ex: Saúde, Exatas, Humanas) um bolsista tende a escolher com base no seu perfil demográfico e ano de ingresso.

Por que é bom para o MBA? Identifica "gargalos" de representatividade. Exemplo: O modelo pode mostrar que certas etnias têm baixíssima probabilidade de prever ingresso em cursos de Alta Tecnologia, gerando um insight para políticas de incentivo.

Algoritmo: Multiclass Classification (XGBoost).

3. Modelo de Tendência Temporal (Time Series Forecasting)
Objetivo: Com dados desde 2005, você pode prever a demanda por bolsas para 2026/2027 por região ou por etnia.

Por que é bom para o MBA? É o modelo mais voltado para "Gestão Estratégica" e planejamento de vagas.

Algoritmo: ARIMA ou Prophet.
